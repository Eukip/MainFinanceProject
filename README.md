# MainFinance

**Personal Finance & Asset Management System**

My personal project for complete control over personal finances, expenses, income, assets, liabilities and all related processes in one place.

I got tired of spreading my financial life across 5–6 different applications. This project is an attempt to build one single tool that covers everything I actually need.

---

## 🎯 Goal of the Project

- Create a convenient all-in-one solution for personal finance management (including self-employed / IP / freelancers)
- Solve my own problem: no existing app has all the features I want in one convenient interface
- Improve my skills as a backend developer through real-world architecture and performance challenges

This is a **personal and educational project** first and foremost. I’m developing it solo.

---

## 🛠 Current Tech Stack

**Backend (monolith):**
- Python 3.12
- FastAPI
- PostgreSQL
